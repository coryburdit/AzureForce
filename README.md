# AzureForce Sync Scripts

Scripts to facilitate faster chagnes to Azure faster.

## adsync.ps1

This script Forces a sync with Azure AD of any changes made this is not an initial sync.

## foceFullSync.ps1

This script Forces a sync with Azure AD of any changes made this is an initial sync.

## Version 1.5 update

Now includes a function to update all domian controllers with the changes as well.
